EmailService

Questo file gestisce l'invio di email utilizzando il modulo `nodemailer`. Viene configurato un trasportatore SMTP con un account Gmail per inviare email. Le credenziali dell'account (email e password) sono caricate tramite variabili d'ambiente dal file `.env`. La funzione `sendEmail` invia un'email con contenuto HTML a un destinatario specificato.

La configurazione del trasportatore include: - Il servizio Gmail - La connessione sicura (SSL) sulla porta 465 - Le credenziali caricate dal file `.env` (utente e password)
